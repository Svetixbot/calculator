# Functional style calculator

## Install and download all the things

Install sbt

	$ brew install sbt

Install ide of choice ?

## Develop

Run this thing

	$ sbt

Compile

	> compile

Run tests

	> test

Run single test

	> test-only com.meow.meow.CatSpec

Run compile in a background

	> ~compile

Run tests in a background

	> ~test
	> ~test-only com.meow.meow.CatSpec
